```js
const promiseLogic = (resolve, reject) => {
  setTimeout(() => {
    resolve('Hello, world!')
  }, 1000)
}

const eventualGreeting = () => new Promise(promiseLogic)

eventualGreeting()
  .then(console.log)
  .catch(console.error)
```

<Notes>

- Whatever you call `resolve` with becomes what the `Promise` resolves with. In this case, a string
